La web "Adoptinder" funciona como una página para buscar una nueva mascota o dar en adopción. La idea es que desde el módulo "Postear" puedas publicar un post acerca de la mascota que estas dando en adopción y desde "Adoptame!" podés enviar una petición sobre la mascota que estás buscando adoptar. Además esta la opción de "Buscar" donde podes filtrar entre los post publicados el tipo de animal que estes buscando.
